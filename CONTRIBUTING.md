# Contributing to Awesome AI Tools

Thank you for your interest in contributing to Awesome AI Tools! This document provides guidelines for contributing to this project.

### Adding a New AI Tool

1. **Fork the Repository**
   - Click the "Fork" button at the top right of the repository page

2. **Clone Your Fork**
   ```bash
   git clone https://github.com/YOUR_USERNAME/awesome-ai-tools.git
   cd awesome-ai-tools
   ```

3. **Create a New Branch**
   ```bash
   git checkout -b add-new-tool
   ```

4. **Add Your Tool**
   - Find the appropriate category in the relevant `.md` file
   - Add your tool following this format:
   ```markdown
   - [Tool Name](https://tool-url.com) - Brief description of what the tool does.. [Pricing]
   ```

5. **Update Both Files**
   - Add the tool to the category-specific file (e.g., `Video.md`, `Image.md`)
   - Also add it to the main `README.md` file in the same category

6. **Commit Your Changes**
   ```bash
   git add .
   git commit -m "Add [Tool Name] to [Category]"
   ```

7. **Push to Your Fork**
   ```bash
   git push origin add-new-tool
   ```

8. **Create a Pull Request**
   - Go to the original repository
   - Click "New Pull Request"
   - Select your branch and submit

### Description Guidelines

- Keep descriptions concise (under 100 characters)
- Focus on the main functionality
- Use clear, simple language
- Avoid marketing jargon
- End with two periods (..)

### Pricing Labels

Use one of these pricing labels:

- `[Free]` - Completely free to use
- `[Freemium]` - Free with paid upgrades
- `[Paid]` - Requires payment
- `[Free Trial]` - Offers a free trial period
- `[Contact for Pricing]` - Pricing not publicly listed
- `[Active deal]` - Currently has a special offer

### Category Placement

Choose the most appropriate category:

- **Productivity Tools**: Personal assistants, research tools, schedulers
- **Text Generators**: Writing, copywriting, paraphrasing
- **Image Tools**: Design, generation, editing
- **Art Generators**: Cartoon, portrait, logo, 3D
- **Video**: Enhancement, editing, generation
- **Audio**: Editing, text-to-speech, music, transcription
- **Business**: Marketing, finance, project management
- **Automation Tools**: Workflows, AI agents
- **Code**: Assistants, low-code/no-code, SQL
- **Miscellaneous**: Fitness, religion, students, fun tools, gaming
